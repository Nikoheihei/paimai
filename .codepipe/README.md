# 产线目录

本目录定义了项目的 AI 开发产线配置。每个产线是一个端到端业务闭环。

## 状态说明

| 状态 | 含义 |
|---|---|
| ✅ 已定义 | 产线配置已写好，可以运行 |
| 🔧 运行中 | 产线正在执行 |
| ⏳ 待定义 | 业务范围清晰，但还没写成产线配置 |
| 📋 规划中 | 知道有这个闭环，但还没拆清楚 |

## 产线清单

| # | 产线 | 状态 | 说明 |
|---|---|---|---|
| 1 | `bid-closed-loop` | ✅ 已定义 | 用户出价 → 原子校验 → 持久化 → 事件推送（已跑完第 1 轮） |
| 2 | `websocket-push` | ⏳ 待定义 | 出价事件 → WebSocket 广播 → 前端实时刷新 |
| 3 | `settle-order` | ⏳ 待定义 | 竞拍结束 → 结算 → 订单生成 → 支付（模拟） |
| 4 | `admin-crud` | ⏳ 待定义 | 商品管理 → 竞拍发布 → 上下架 |
| 5 | `h5-bidding-ui` | ⏳ 待定义 | 移动端竞拍页 → 出价交互 → 实时状态 |
| 6 | `admin-dashboard` | ⏳ 待定义 | PC 后台 → 竞拍管理 → 订单管理 |
| 7 | `load-testing` | ⏳ 待定义 | 压测脚本 → 并发出价 → 一致性验证 |
