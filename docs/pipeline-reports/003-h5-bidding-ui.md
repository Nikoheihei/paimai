## 一、本次生成内容

### 1.1 背景

H5 前端之前是 Vite 脚手架页面，空有组件骨架但缺少完整的出价交互。现在改造成可以实际出价、看到实时更新的竞拍直播间。

### 1.2 修改的文件

| 文件 | 改动 |
|---|---|
| `web-h5/src/api/client.ts` | 新增 `placeBid` 函数（POST 出价 + 错误处理） |
| `web-h5/src/components/AuctionPanel.tsx` | 完整改写：倒计时、快捷出价按钮、自定义金额输入、出价状态反馈、被超越提醒、排行榜"我"的高亮 |
| `web-h5/src/App.css` | 追加倒计时动画、出价面板、成交信息、排名高亮样式 |
| `web-h5/src/pages/RoomPage.tsx` | 补传 `userId` prop |

### 1.4 交互流程

```
页面加载
  ↓
getRoomAuctions(roomId, 'running') → 获取当前竞拍
  ↓
getAuction(id) + getRanking(id) → 加载详情和排行榜
  ↓
useWebSocket(roomId, userId) → 连接 WS
  ↓
用户点快捷出价 / 输入金额 → placeBid()
  ↓
后端处理 → XADD Stream → WS 广播 bid.accepted
  ↓
前端收到消息 → 自动刷新价格 + 排行榜
```

## 四、✅ 已处理的边界情况

| 边界 | 处理方式 |
|---|---|
| 倒计时更新 | `setInterval` 每秒刷新，竞拍结束自动停 |
| 倒计时紧急态 | < 60 秒时数字变红 + 脉冲动画 |
| 快捷出价 | 3 个按钮（+1x / +2x / +5x 加价幅度） |
| 出价状态反馈 | idle → sending → ok/fail，带具体中文提示 |
| 被超越提醒 | WS 收到消息（后续扩展 `outbid` 类型时触发） |
| 我的排名高亮 | 排行榜中 userId 匹配时标记"我"+ 高亮背景 |
| 成交展示 | 封顶价到达时显示成交价 + 赢家 + 绿色背景 |
| 出价金额校验 | 非数字或 ≤ 0 时本地拦截 |
